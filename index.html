üìã Setup Instructions:

Step 1: Create Google Cloud Project
  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable the "Google Sheets API"
  4. Create API credentials (API Key)
Step 2: Set Up Your Spreadsheet
  1. Create a new Google Sheets document
  2. Set up columns: Seat ID, Student Name, Department, Session, Room No, Building, Floor No, Floor Teacher, Status
  3. Share the spreadsheet (Anyone with the link can view)
  4. Copy the spreadsheet ID from the URL
Step 3: Update Configuration

Edit the index.html file and update these values:

const GOOGLE_SHEETS_CONFIG = {
    apiKey: 'YOUR_API_KEY_HERE',
    spreadsheetId: 'YOUR_SPREADSHEET_ID_HERE',
    range: 'Sheet1!A:I',
    enabled: true  // Change to true
};
